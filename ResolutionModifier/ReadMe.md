ResolutionModifier（camera像素化/高清化）
----------

**u3d version**: 2019.3.4f1



## 用途

借助render texture和raw image，通过调整render texture尺寸达成像素画/高清化效果。

## 脚本使用方式

1. 将脚本挂在任意物体下

2. 新建raw image，最终处理后图像将显示在该raw image上

3. 将camera、raw image挂在脚本上
